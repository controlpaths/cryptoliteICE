# hardware/

Hardware documentation for the CryptoLite-ICE USB stick.

| File | Description |
|------|-------------|
| `cryptoliteICE_sch.pdf` | Full schematic of the board (iCE40UP5K + RP2040, power, USB, status LEDs and the shared SPI bus). |

Only the schematic is published here. The complete KiCad project (PCB layout,
gerbers, BOM and production files) is kept private. The 3D board renders shown
in the top-level README live in [`../doc/`](../doc/).

Refer to the top-level README for the component reference table, the power
architecture, and the RP2040 <-> iCE40 signal map.
